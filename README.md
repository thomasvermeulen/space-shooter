# Space Shooter





## Installatie

Spel spelen

```bash
• Zorg dat Python en PyGame zijn geïnstalleerd. 
• Download de volledige file via Github en unzip deze naar je desktop (of plek naar keuze)
• Open de file via een IDE naar keuze.
• Run level1.py

```
